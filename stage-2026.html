Entreprise

EURL Szlazak

Troisième stage de BTS SIO, effectué du 12 janvier 2026 au 20 février 2026 autour d'un besoin concret de digitalisation.

Axes

Trois objectifs clairs

  • Développement web
  • Digitalisation des outils de l'entreprise
  • Amélioration de l'organisation interne

Mission

Solutions numériques pour l'entreprise

Ma mission principale a été de développer un site web vitrine et une application web de gestion interne pour optimiser le suivi des chantiers, des équipes et des ressources.

Tâches réalisées

  • Création d'un site web vitrine en HTML, CSS et JavaScript
  • Mise en ligne du site via Netlify
  • Développement d'une application web avec Laravel
  • Création de modules pour les chantiers, le stockage, les équipes, l'agenda et les fiches d'heures
  • Mise en place de l'authentification et de la gestion des rôles
  • Déploiement sur o2switch, base de données et résolution de problèmes techniques

Compétences techniques

  • HTML, CSS, JavaScript et PHP
  • Framework Laravel
  • Base de données MySQL
  • Déploiement via FTP et cPanel
  • Architecture MVC, routes et contrôleurs
  • Authentification, rôles et sessions

Compétences transversales

  • Autonomie dans la recherche de solutions
  • Adaptation face aux problèmes techniques
  • Organisation du travail
  • Compréhension des besoins d'un client professionnel
  • Communication avec le chef d'entreprise

Bilan

Progression confirmée

Ce stage m'a permis de renforcer mes compétences techniques, de gagner en autonomie et en rigueur, et de confirmer mon intérêt pour le développement web dans un cadre professionnel concret.
